A bit about me

I'm Sam, a computer science student and software engineer interested in programming language theory, algorithms, and product development. I study at the University of Southern California, and have worked on numerous projects including developing mobile platforms at Glia Mental Health, building interfaces for all electric semi-trucks, and providing API support for Messenger Stories!

Previous Work

Here are some of the things I've done previously:

Glia Mental Health: I was the primary developer for Glia Mental Health. I helped develop their mobile platform, as well as their web presence while competing in the Min Family Challenge at USC.

Thor Trucks: I worked on building a web presence for Thor Trucks, a company trying to build a fully electric semi-truck. I also served to build the user interface for the inside of the vehicle for the primary control panel.

Facebook Inc: I helped extend API support for new features coming into Messenger Stories, as well as driving a migration of legacy systems for Messenger.com. I also improved visibility for stories features on Messenger iOS.

Google LLC: I'm currently with Google under Chrome, working on chrome enterprise and educational management!
